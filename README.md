\# Resume Evaluation System (NLP-Based)



This project uses Natural Language Processing (NLP) to analyze resumes and match them with job descriptions.



\## Features

\- Extracts skills, experience, and education

\- Calculates similarity score using TF-IDF

\- Ranks resumes based on relevance



\## Technologies

\- Python

\- SpaCy / NLTK

\- Scikit-learn

\- Streamlit



\## Run

python app.py
